Android Java 1.7 环境配置

下载jdk 1.7 https://repo.huaweicloud.com/java/jdk/7u80-b15/jdk-7u80-linux-x64.tar.gz 解压 tar xzvf jdk-7u80-linux-x64.tar.gz android 18.04 jdk1.7 android 7.1 安装 sudo mv jdk1.7.0_80 /usr/lib/ switchjdk7.sh 文件内容如下，该文件放在代码顶层目录： # add JDK7 cofig ---- export JAVA_HOME=/usr/lib/jdk1.7.0_80/ export JRE_HOME=${JAVA_HOME}/jre export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib export PATH=${JAVA_HOME}/bin:$PATH # add JDK7 cofig ---- 执行如下命令切换java 1.7，仅当前终端有效 source switchjdk7.sh https://repo.huaweicloud.com/java/jdk/6u45-b06/ https://repo.huaweicloud.com/java/jdk/7u80-b15/ https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html
