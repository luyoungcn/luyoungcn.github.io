рҹ“ҳLinux еҲқе§ӢеҢ–зі»з»ҹжҠҖжңҜж–ҮжЎЈпјҡд»Һ SysVinit еҲ° systemd зҡ„жј”иҝӣ

September 18, 2025
#ubuntu
#systemd
#SysVinit
#еҲқе§ӢеҢ–зі»з»ҹ

Linux еҲқе§ӢеҢ–зі»з»ҹжҠҖжңҜж–ҮжЎЈпјҡд»Һ SysVinit еҲ° systemd зҡ„жј”иҝӣ

1. жҰӮиҝ°

Linux еҲқе§ӢеҢ–зі»з»ҹжҳҜж“ҚдҪңзі»з»ҹеҗҜеҠЁиҝҮзЁӢдёӯз¬¬дёҖдёӘз”ЁжҲ·з©әй—ҙиҝӣзЁӢпјҲPID 1пјүпјҢиҙҹиҙЈеј•еҜјз”ЁжҲ·з©әй—ҙзҺҜеўғгҖҒз®ЎзҗҶзі»з»ҹжңҚеҠЎе’ҢиҝӣзЁӢгҖӮжң¬ж–ҮжЎЈиҜҰз»Ҷд»Ӣз»ҚдәҶ Linux еҲқе§ӢеҢ–зі»з»ҹзҡ„еҸ‘еұ•еҺҶзЁӢгҖҒжҠҖжңҜзү№зӮ№еҸҠеҗ„зі»з»ҹзҡ„жҜ”иҫғгҖӮ

2. еҲқе§ӢеҢ–зі»з»ҹеҸ‘еұ•еҺҶзЁӢ

3. дё»иҰҒеҲқе§ӢеҢ–зі»з»ҹиҜҰи§Ј

3.1 SysVinit (System V Init)

вҖӢвҖӢиҜһз”ҹж—¶й—ҙвҖӢвҖӢпјҡ1980е№ҙд»ЈпјҲжәҗиҮӘUNIX System Vпјү

вҖӢвҖӢж ёеҝғзү№жҖ§вҖӢвҖӢпјҡ

  • еҹәдәҺиҝҗиЎҢзә§еҲ«пјҲ0-6пјүзҡ„зі»з»ҹзҠ¶жҖҒз®ЎзҗҶ
  • дёІиЎҢеҗҜеҠЁиҝҮзЁӢпјҢйЎәеәҸжү§иЎҢеҲқе§ӢеҢ–и„ҡжң¬
  • дҪҝз”ЁShellи„ҡжң¬з®ЎзҗҶжңҚеҠЎеҗҜеҠЁ/еҒңжӯў

вҖӢвҖӢи„ҡжң¬з»“жһ„вҖӢвҖӢпјҡ

/etc/rc.d/
в”ңв”Җв”Җ rc0.d/   # е…іжңәиҝҗиЎҢзә§еҲ«
в”ңв”Җв”Җ rc1.d/   # еҚ•з”ЁжҲ·жЁЎејҸ
в”ңв”Җв”Җ rc2.d/   # еӨҡз”ЁжҲ·ж— зҪ‘з»ң
в”ңв”Җв”Җ rc3.d/   # еӨҡз”ЁжҲ·жҺ§еҲ¶еҸ°
в”ңв”Җв”Җ rc4.d/   # з”ЁжҲ·иҮӘе®ҡд№ү
в”ңв”Җв”Җ rc5.d/   # еӨҡз”ЁжҲ·еӣҫеҪўз•Ңйқў
в”ңв”Җв”Җ rc6.d/   # йҮҚеҗҜиҝҗиЎҢзә§еҲ«
в””в”Җв”Җ init.d/  # е®һйҷ…и„ҡжң¬зӣ®еҪ•

вҖӢвҖӢдё»иҰҒе‘Ҫд»ӨвҖӢвҖӢпјҡ

# з®ЎзҗҶжңҚеҠЎ
/etc/init.d/service_name start|stop|restart|status

# ж”№еҸҳиҝҗиЎҢзә§еҲ«
init 3
telinit 5

# е…іжңә/йҮҚеҗҜ
shutdown -h now
reboot

вҖӢвҖӢдјҳзјәзӮ№вҖӢвҖӢпјҡ

  • вң… з®ҖеҚ•зЁіе®ҡпјҢжҳ“дәҺзҗҶи§Је’Ңи°ғиҜ•
  • вң… и„ҡжң¬йҖҸжҳҺпјҢеҸҜе®Ңе…ЁиҮӘе®ҡд№ү
  • вқҢ еҗҜеҠЁйҖҹеәҰж…ўпјҲдёІиЎҢжү§иЎҢпјү
  • вқҢ ж— жі•еӨ„зҗҶеҠЁжҖҒдәӢд»¶пјҲеҰӮзғӯжҸ’жӢ”и®ҫеӨҮпјү
  • вқҢ дҫқиө–е…ізі»з®ЎзҗҶеӨҚжқӮ

3.2 Upstart

вҖӢвҖӢиҜһз”ҹж—¶й—ҙвҖӢвҖӢпјҡ2006е№ҙпјҲз”ұUbuntuејҖеҸ‘пјү

вҖӢвҖӢж ёеҝғеҲӣж–°вҖӢвҖӢпјҡ

  • дәӢд»¶й©ұеҠЁжһ¶жһ„пјҢе“Қеә”зі»з»ҹдәӢд»¶
  • е№¶иЎҢжңҚеҠЎеҗҜеҠЁпјҢжҳҫи‘—жҸҗеҚҮеҗҜеҠЁйҖҹеәҰ
  • жӣҙеҘҪзҡ„зЎ¬д»¶зғӯжҸ’жӢ”ж”ҜжҢҒ

вҖӢвҖӢй…ҚзҪ®ж–Үд»¶зӨәдҫӢвҖӢвҖӢпјҲ/etc/init/nginx.confпјүпјҡ

# Nginx - й«ҳжҖ§иғҪHTTPжңҚеҠЎеҷЁ
description "Nginx HTTPжңҚеҠЎеҷЁ"

# еңЁиҝҗиЎҢзә§еҲ«2гҖҒ3гҖҒ4гҖҒ5еҗҜеҠЁпјҢеңЁ0гҖҒ1гҖҒ6еҒңжӯў
start on runlevel [2345]
stop on runlevel [016]

# жңҹжңӣд»Ҙdaemonж–№ејҸиҝҗиЎҢ
expect fork

# еҗҜеҠЁжңҚеҠЎ
exec /usr/sbin/nginx -g "daemon on;"

вҖӢвҖӢдё»иҰҒе‘Ҫд»ӨвҖӢвҖӢпјҡ

# з®ЎзҗҶд»»еҠЎ
start nginx
stop nginx
status nginx

# жҹҘзңӢдәӢд»¶
initctl list
initctl emit event-name

вҖӢвҖӢеә”з”Ёжғ…еҶөвҖӢвҖӢпјҡ

  • жӣҫз”ЁдәҺUbuntuпјҲ9.10-14.10пјүгҖҒRHEL 6зӯү
  • зҺ°е·Іиў«systemdеҸ–д»Ј

3.3 OpenRC

вҖӢвҖӢиҜһз”ҹж—¶й—ҙвҖӢвҖӢпјҡ2007е№ҙпјҲз”ұGentooзӨҫеҢәејҖеҸ‘пјү

вҖӢвҖӢи®ҫи®Ўзү№зӮ№вҖӢвҖӢпјҡ

  • дҫқиө–й©ұеҠЁзҡ„еҲқе§ӢеҢ–зі»з»ҹ
  • е…је®№дј з»ҹSysVinitи„ҡжң¬
  • дёҚдҫқиө–зү№е®ҡLinuxеҶ…ж ёзү№жҖ§

вҖӢвҖӢжңҚеҠЎж–Үд»¶зӨәдҫӢвҖӢвҖӢпјҲ/etc/init.d/sshdпјүпјҡ

#!/sbin/openrc-run

description="OpenSSH е®ҲжҠӨиҝӣзЁӢ"

command="/usr/sbin/sshd"
command_args="-D"
pidfile="/var/run/sshd.pid"

depend() {
    need net
    use dns
    before firewall
}

вҖӢвҖӢдё»иҰҒе‘Ҫд»ӨвҖӢвҖӢпјҡ

# з®ЎзҗҶжңҚеҠЎ
rc-service service_name start
rc-status
rc-update add service_name default

вҖӢвҖӢеә”з”Ёжғ…еҶөвҖӢвҖӢпјҡ

  • Gentoo LinuxеҸҠе…¶иЎҚз”ҹзүҲ
  • Alpine LinuxпјҲDockerеҹәзЎҖй•ңеғҸеёёз”Ёпјү

3.4 runit

вҖӢвҖӢиҜһз”ҹж—¶й—ҙвҖӢвҖӢпјҡ2004е№ҙ

вҖӢвҖӢи®ҫи®Ўе“ІеӯҰвҖӢвҖӢпјҡ

  • з®ҖеҚ•гҖҒеҸҜйқ гҖҒжңҖе°ҸеҢ–и®ҫи®Ў
  • ж ёеҝғеҠҹиғҪпјҡиҝӣзЁӢзӣ‘зқЈпјҲиҮӘеҠЁйҮҚеҗҜеҙ©жәғжңҚеҠЎпјү
  • жһҒеҝ«зҡ„еҗҜеҠЁйҖҹеәҰ

вҖӢвҖӢзӣ®еҪ•з»“жһ„вҖӢвҖӢпјҡ

/etc/service/
в””в”Җв”Җ nginx/
    в”ңв”Җв”Җ run      # еҗҜеҠЁи„ҡжң¬
    в””в”Җв”Җ finish   # еҒңжӯўи„ҡжң¬

вҖӢвҖӢжңҚеҠЎзӨәдҫӢвҖӢвҖӢпјҲ/etc/service/nginx/runпјүпјҡ

#!/bin/sh
exec 2>&1
exec /usr/sbin/nginx -g "daemon off;"

вҖӢвҖӢдё»иҰҒе‘Ҫд»ӨвҖӢвҖӢпјҡ

# з®ЎзҗҶжңҚеҠЎ
sv start nginx
sv status nginx
sv restart nginx

вҖӢвҖӢеә”з”Ёжғ…еҶөвҖӢвҖӢпјҡ

  • Void Linuxй»ҳи®ӨеҲқе§ӢеҢ–зі»з»ҹ
  • иҪ»йҮҸзә§е®№еҷЁзҺҜеўғ
  • йңҖиҰҒй«ҳеҸҜйқ жҖ§зҡ„жңҚеҠЎзҺҜеўғ

3.5 systemd

вҖӢвҖӢиҜһз”ҹж—¶й—ҙвҖӢвҖӢпјҡ2010е№ҙпјҲз”ұLennart Poetteringе’ҢKay SieversејҖеҸ‘пјү

вҖӢвҖӢжһ¶жһ„еҲӣж–°вҖӢвҖӢпјҡ

  • е№¶иЎҢеҗҜеҠЁжңҚеҠЎпјҢеҹәдәҺдҫқиө–е…ізі»и§Јжһҗ
  • з»ҹдёҖзҡ„жңҚеҠЎз®ЎзҗҶй…ҚзҪ®пјҲеҚ•е…ғж–Үд»¶пјү
  • йӣҶжҲҗзі»з»ҹз®ЎзҗҶеҠҹиғҪпјҲж—Ҙеҝ—гҖҒе®ҡж—¶д»»еҠЎгҖҒзҪ‘з»ңзӯүпјү

вҖӢвҖӢжңҚеҠЎеҚ•е…ғзӨәдҫӢвҖӢвҖӢпјҲ/etc/systemd/system/nginx.serviceпјүпјҡ

[Unit]
Description=The NGINX HTTP and reverse proxy server
After=network.target network-online.target
Wants=network-online.target
Documentation=https://nginx.org/en/docs/

[Service]
Type=forking
PIDFile=/var/run/nginx.pid
ExecStartPre=/usr/bin/nginx -t
ExecStart=/usr/bin/nginx
ExecReload=/bin/kill -s HUP $MAINPID
ExecStop=/bin/kill -s QUIT $MAINPID
TimeoutStopSec=5
KillMode=mixed

[Install]
WantedBy=multi-user.target

вҖӢвҖӢдё»иҰҒе‘Ҫд»ӨвҖӢвҖӢпјҡ

# з®ЎзҗҶжңҚеҠЎ
systemctl start nginx
systemctl status nginx
systemctl enable nginx

# зі»з»ҹз®ЎзҗҶ
systemctl halt        # е…іжңә
systemctl reboot      # йҮҚеҗҜ
systemctl suspend     # жҢӮиө·

# ж—Ҙеҝ—жҹҘзңӢ
journalctl -u nginx
journalctl -f

# еҲҶжһҗеҗҜеҠЁжҖ§иғҪ
systemd-analyze blame
systemd-analyze critical-chain nginx.service

вҖӢвҖӢеә”з”Ёжғ…еҶөвҖӢвҖӢпјҡ

  • з»қеӨ§еӨҡж•°дё»жөҒLinuxеҸ‘иЎҢзүҲй»ҳи®ӨеҲқе§ӢеҢ–зі»з»ҹ
  • DebianгҖҒUbuntuгҖҒRHELгҖҒCentOSгҖҒFedoraгҖҒArch Linuxзӯү

3.6 дё“жңүйўҶеҹҹж–№жЎҲ

вҖӢвҖӢprocdвҖӢвҖӢпјҡ

  • з”ЁдәҺOpenWrtи·Ҝз”ұеҷЁзҡ„иҪ»йҮҸзә§еҲқе§ӢеҢ–зі»з»ҹ
  • дё“жіЁдәҺеөҢе…ҘејҸи®ҫеӨҮиө„жәҗйҷҗеҲ¶
  • ж ёеҝғеҠҹиғҪпјҡиҝӣзЁӢз®ЎзҗҶе’Ңзӣ‘жҺ§

вҖӢвҖӢs6вҖӢвҖӢпјҡ

  • еҸҰдёҖж¬ҫиҪ»йҮҸзә§иҝӣзЁӢзӣ‘зқЈе·Ҙе…·йӣҶ
  • и®ҫи®ЎжіЁйҮҚз®ҖеҚ•жҖ§е’ҢжӯЈзЎ®жҖ§
  • еёёз”ЁдәҺе®№еҷЁе’ҢжңҖе°ҸеҢ–зі»з»ҹ

4. жҠҖжңҜеҜ№жҜ”еҲҶжһҗ

4.1 жҖ§иғҪеҜ№жҜ”

зү№жҖ§ SysVinit Upstart OpenRC runit systemd
еҗҜеҠЁйҖҹеәҰ ж…ў дёӯзӯү дёӯзӯү еҝ« еҫҲеҝ«
иө„жәҗеҚ з”Ё дҪҺ дёӯзӯү дҪҺ еҫҲдҪҺ дёӯзӯү
е№¶иЎҢиғҪеҠӣ ж—  жңүйҷҗ жңүйҷҗ жңү е…Ёйқў

4.2 еҠҹиғҪзү№жҖ§еҜ№жҜ”

еҠҹиғҪ SysVinit Upstart OpenRC runit systemd
дҫқиө–з®ЎзҗҶ жүӢеҠЁ дәӢд»¶еҹәзЎҖ дҫқиө–еҹәзЎҖ з®ҖеҚ• иҮӘеҠЁдҫқиө–и§Јжһҗ
жңҚеҠЎзӣ‘жҺ§ ж—  жңүйҷҗ жңүйҷҗ жңү жңү
ж—Ҙеҝ—йӣҶжҲҗ ж—  ж—  ж—  ж—  жңү(journald)
зғӯжҸ’жӢ”ж”ҜжҢҒ жңүйҷҗ жңү жңүйҷҗ ж—  жңү
е®№еҷЁж”ҜжҢҒ ж—  жңүйҷҗ жңүйҷҗ жңү жңү

4.3 е…је®№жҖ§еҜ№жҜ”

ж–№йқў SysVinit Upstart OpenRC runit systemd
дј з»ҹи„ҡжң¬е…је®№ е®Ңе…Ё йғЁеҲҶ е®Ңе…Ё йңҖиҰҒйҖӮй…Қ йғЁеҲҶ
и·ЁеҸ‘иЎҢзүҲж”ҜжҢҒ е№ҝжіӣ жңүйҷҗ дёӯзӯү жңүйҷҗ е№ҝжіӣ
й…ҚзҪ®еӨҚжқӮеәҰ дҪҺ дёӯзӯү дёӯзӯү дҪҺ й«ҳ

5. иҝҒз§»жҢҮеҚ—

5.1 д»ҺSysVinitиҝҒз§»еҲ°systemd

вҖӢжңҚеҠЎи„ҡжң¬иҪ¬жҚўвҖӢвҖӢпјҡ

  • е°Ҷ/etc/init.d/и„ҡжң¬иҪ¬жҚўдёә.serviceеҚ•е…ғж–Үд»¶
  • дҪҝз”Ёsystemd-analyzeжЈҖжҹҘеҗҜеҠЁжҖ§иғҪ

вҖӢвҖӢиҝҗиЎҢзә§еҲ«жҳ е°„вҖӢвҖӢпјҡ

SysVinitиҝҗиЎҢзә§еҲ« systemdзӣ®ж Ү
0 poweroff.target
1 rescue.target
2, 3, 4 multi-user.target
5 graphical.target
6 reboot.target

вҖӢеёёз”Ёе‘Ҫд»ӨеҜ№жҜ”вҖӢвҖӢпјҡ

SysVinitе‘Ҫд»Ө systemdе‘Ҫд»Ө
service ssh start systemctl start ssh
chkconfig ssh on systemctl enable ssh
runlevel systemctl get-default

5.2 д»ҺsystemdиҝҒз§»еҲ°runit

вҖӢеҲӣе»әжңҚеҠЎзӣ®еҪ•вҖӢвҖӢпјҡ

mkdir -p /etc/sv/nginx

вҖӢеҲӣе»әrunи„ҡжң¬вҖӢвҖӢпјҡ

cat > /etc/sv/nginx/run << EOF
#!/bin/sh
exec /usr/sbin/nginx -g "daemon off;"
EOF
chmod +x /etc/sv/nginx/run

вҖӢеҲӣе»әжңҚеҠЎй“ҫжҺҘвҖӢвҖӢпјҡ

ln -s /etc/sv/nginx /var/service/

6. йҖүжӢ©е»әи®®

6.1 ж №жҚ®дҪҝз”ЁеңәжҷҜйҖүжӢ©

вҖӢвҖӢжңҚеҠЎеҷЁзҺҜеўғвҖӢвҖӢпјҡ

  • вң… systemdпјҡеҠҹиғҪе…ЁйқўпјҢз®ЎзҗҶж–№дҫҝпјҢз”ҹжҖҒдё°еҜҢ
  • вң… OpenRCпјҡзЁіе®ҡеҸҜйқ пјҢиө„жәҗеҚ з”ЁдҪҺ

вҖӢвҖӢжЎҢйқўзҺҜеўғвҖӢвҖӢпјҡ

  • вң… systemdпјҡдёҺзҺ°д»ЈжЎҢйқўйӣҶжҲҗеәҰй«ҳ
  • вң… runitпјҡиҪ»йҮҸеҝ«йҖҹпјҢе“Қеә”иҝ…йҖҹ

вҖӢвҖӢеөҢе…ҘејҸи®ҫеӨҮвҖӢвҖӢпјҡ

  • вң… procdпјҡдё“дёәиө„жәҗеҸ—йҷҗзҺҜеўғи®ҫи®Ў
  • вң… runitпјҡз®ҖеҚ•еҸҜйқ пјҢеҚ з”Ёиө„жәҗжһҒе°‘

вҖӢвҖӢе®№еҷЁзҺҜеўғвҖӢвҖӢпјҡ

  • вң… runitпјҡиҪ»йҮҸзә§пјҢйҖӮеҗҲжңҖе°ҸеҢ–й•ңеғҸ
  • вң… s6пјҡдё“жіЁдәҺиҝӣзЁӢзӣ‘зқЈ

вҖӢвҖӢдј з»ҹзі»з»ҹз»ҙжҠӨвҖӢвҖӢпјҡ

  • вң… SysVinitпјҡе…је®№иҖҒж—§зі»з»ҹ
  • вң… OpenRCпјҡе№іиЎЎдј з»ҹдёҺзҺ°д»ЈйңҖжұӮ

6.2 жңӘжқҘеҸ‘еұ•иҖғиҷ‘

  1. вҖӢвҖӢsystemdвҖӢвҖӢз»§з»ӯеҚ жҚ®дё»жөҒең°дҪҚпјҢеҠҹиғҪдёҚж–ӯеўһејә
  2. иҪ»йҮҸзә§ж–№жЎҲеңЁе®№еҷЁе’ҢеөҢе…ҘејҸйўҶеҹҹдҝқжҢҒйҮҚиҰҒең°дҪҚ
  3. е®үе…ЁжҖ§гҖҒе®№еҷЁйӣҶжҲҗе’Ңиө„жәҗжҺ§еҲ¶жҲҗдёәеҸ‘еұ•йҮҚзӮ№

7. жҖ»з»“

LinuxеҲқе§ӢеҢ–зі»з»ҹз»ҸеҺҶдәҶд»Һз®ҖеҚ•зҡ„дёІиЎҢеҗҜеҠЁеҲ°еӨҚжқӮзҡ„е№¶иЎҢдҫқиө–з®ЎзҗҶзҡ„жј”иҝӣиҝҮзЁӢгҖӮеҪ“еүҚsystemdе·ІжҲҗдёәдё»жөҒйҖүжӢ©пјҢдҪҶиҪ»йҮҸзә§ж–№жЎҲеҰӮrunitгҖҒOpenRCеңЁзү№е®ҡеңәжҷҜдёӢд»Қжңүе…¶д»·еҖјгҖӮйҖүжӢ©еҲқе§ӢеҢ–зі»з»ҹеә”иҖғиҷ‘е…·дҪ“йңҖжұӮпјҡеҠҹиғҪе®Ңж•ҙжҖ§гҖҒиө„жәҗйҷҗеҲ¶гҖҒе…је®№жҖ§иҰҒжұӮзӯүеӣ зҙ гҖӮ

зҗҶи§Јеҗ„еҲқе§ӢеҢ–зі»з»ҹзҡ„и®ҫи®Ўе“ІеӯҰе’Ңе®һзҺ°зү№зӮ№пјҢжңүеҠ©дәҺеңЁдёҚеҗҢеңәжҷҜдёӢеҒҡеҮәеҗҲйҖӮзҡ„жҠҖжңҜйҖүеһӢпјҢе№¶жңүж•ҲиҝӣиЎҢзі»з»ҹз»ҙжҠӨе’Ңж•…йҡңжҺ’йҷӨгҖӮ
