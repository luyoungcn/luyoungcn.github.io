黑芝麻平台交叉编译 openssl

前置条件

编译环境：黑芝麻 Linux23 Docker 开发环境，已经默认配置了交叉编译环境。

源码下载

wget https://www.openssl.org/source/openssl-1.1.1k.tar.gz
tar -xzvf openssl-1.1.1k.tar.gz
cd openssl-1.1.1k

配置

准备目标文件安装目录：

mkdir _INSTALL

查看帮助手册：
